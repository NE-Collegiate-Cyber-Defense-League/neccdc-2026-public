## Org Chart
Make sure employee_id & manager_id are populated

Convert the yaml into a CSV. [Online tool](https://convertcsv.com/yaml-to-csv.htm)

I used [Lucid](https://help.lucid.co/hc/en-us/articles/44818725952276-How-do-I-import-an-org-chart-from-a-CSV-Excel-file-or-Google-Sheet) to generate the diagram chart based on the csv. Make sure to hide the password & totp codes.
